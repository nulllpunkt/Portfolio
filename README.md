# Terminal Portfolio

A minimalist, old-school computer terminal portfolio for a creative writer and photographer. It features an interactive command-line interface directly in the browser using plain HTML, CSS, and JS.

## Getting Started

1. Open `index.html` in your favorite web browser.
2. Alternatively, if you have an extension like **Live Server** in VS Code, right-click on `index.html` and choose "Open with Live Server".
   
## Available Commands
- `help`: See the list of all commands.
- `whoami`: Read the creator's bio.
- `works`: Get a list of creative portfolios and collections.
- `poetry` / `prose` / `photography`: Specialized commands to view particular art forms.
- `contact`: View contact or social media information.
- `clear`: Clear the terminal screen.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
